# 🎂 Pixel-Art Birthday Café

A cozy, retro-RPG style birthday surprise website.

## ✨ Features

- **Interactive Envelope:** Click to open a greeting card with smooth slide animations.
- **Café Scene:** A beautifully detailed pixel-art café with animated characters.
- **Gift Surprise:** A interactive gift box that pops open to reveal prizes (Flowers, Chocolate, Gaming Mouse, and a Laptop Stand!).
- **Dynamic Dialogue:** A narrative-driven experience with a "Continue" conversation system.
- **Personalized:** Custom banner for Dhrisit(Kuku).

## 🚀 Deployment

This project is ready to be deployed on **Vercel**.

1. Create a new repository on [GitHub](https://github.com/new).
2. Run the following commands in this directory:
   ```bash
   git add .
   git commit -m "Initial commit for deployment"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```
3. Go to [Vercel](https://vercel.com/new) and import the repository.
4. Vercel will automatically detect it as a static project and deploy it!
